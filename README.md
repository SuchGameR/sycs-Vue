# SYCS
このアプリケーションはスレッド型のコミュニケーションシステムを提供します。

実装するものは以下の通りです。

## 最も重要な実装機能
・アカウント周りの関連機能
    ・ログイン機能
    ・サインアップ機能
    ・ログアウト機能
    ・アカウント設定機能
    ・アカウント削除機能
    ・アカウント検索機能
    ・アカウントID重複不可機能

・スレッド周りの関連機能
    ・スレッドへの書き込み機能
    ・書き込みの削除機能
    ・スレッド作成機能
    ・スレッド削除機能
    ・スレッド編集機能
    ・スレッド一覧表示
    ・スレッド検索機能
    ・ファイルのアップロード（添付、画像、動画）機能
    ・ファイルの削除機能
    ・メンション機能
    ・URL短縮機能

・その他の機能
    ・テーマ変更
    ・通知機能
